# Phase 2 性能测试

本目录包含 Phase 2 核心框架层的性能测试。

## 测试目标

### 响应时间基准
- 登录操作：< 100ms
- 权限检查：< 10ms
- 菜单加载：< 50ms
- CRUD 操作：< 200ms

### 并发性能
- 支持 100+ 并发用户
- 连接池无阻塞
- 无死锁

### 资源使用
- 内存占用合理
- CPU 使用率正常
- 连接正确释放

## 测试场景

### 1. 连接池性能测试
```
- 创建 100 个并发连接
- 测试连接获取/释放性能
- 验证连接池大小限制
```

### 2. 元数据缓存测试
```
- 首次加载时间
- 缓存命中响应时间
- 内存占用
```

### 3. 权限检查性能测试
```
- 单次权限检查时间
- 批量权限检查时间
- 缓存效果验证
```

### 4. 菜单树性能测试
```
- 1000+ 菜单项加载时间
- 权限过滤性能
- 树形结构构建时间
```

### 5. CRUD 操作性能测试
```
- 查询 1000 条记录时间
- 分页查询性能
- 批量操作性能
```

## 测试工具

- DUnitX Benchmark
- AQTime（可选）
- Manual Performance Testing
