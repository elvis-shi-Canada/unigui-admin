## 2. 测试用例

### 2.1 单元测试 (UniPluginTest.pas)

#### TestPluginRegistration
- **描述:** 测试插件注册功能
- **验证点:**
  - [ ] 插件能正确注册到注册表
  - [ ] 注册后能通过 PluginID 查找插件
  - [ ] 插件信息正确存储
  - [ ] 重复注册抛出异常

#### TestPluginInitialization
- **描述:** 测试插件初始化功能
- **验证点:**
  - [ ] 新创建插件状态为 psUninitialized
  - [ ] Initialize 方法正确执行
  - [ ] 初始化后状态变为 psInitialized
  - [ ] 初始化失败时状态变为 psError

#### TestPluginActivation
- **描述:** 测试插件激活功能
- **验证点:**
  - [ ] Activate 方法正确执行
  - [ ] 激活后状态变为 psActivated
  - [ ] 未初始化时激活抛出异常
  - [ ] 权限检查功能正常

#### TestDependencyValidation
- **描述:** 测试依赖验证功能
- **验证点:**
  - [ ] 能正确识别依赖关系
  - [ ] 能检测缺失的依赖插件
  - [ ] 依赖关系正确存储在图中
  - [ ] GetDependencies 返回正确的依赖列表

#### TestCircularDependencyDetection
- **描述:** 测试循环依赖检测功能
- **验证点:**
  - [ ] 能正确检测循环依赖
  - [ ] 返回正确的循环路径
  - [ ] CalculateLoadOrder 在循环时抛出异常
  - [ ] 无循环依赖时能正确计算加载顺序
