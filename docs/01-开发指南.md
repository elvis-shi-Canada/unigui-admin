## 开发环境搭建

### 1. 系统要求

- **操作系统**: Windows 10/11 或 Windows Server 2016+
- **开发工具**: Delphi 12 Athens 或更高版本
- **数据库**: SQL Server 2016+ (Express/Standard/Enterprise)
- **浏览器**: Chrome/Edge (用于调试)
- **内存**: 最低 8GB，推荐 16GB+

### 2. 安装 Delphi 组件

#### 必需组件
1. **UniGUI 1.6+**
   - 从 FMSoft 官网下载安装包
   - 安装到 Delphi IDE
   - 验证安装：File → New → Other → UniGUI for Delphi

2. **FireDAC**
   - Delphi 自带，确保在 IDE 中启用
   - 验证：Tools → Palette → 搜索 TFDConnection

3. **DUnitX**
   - Delphi 自带单元测试框架
   - 验证：Tools → Test Runner

#### 可选组件
- **UniChart**: 图表组件
- **UniDBGrid**: 高级数据网格
- **UniURLFrame**: URL 框架组件

### 3. 数据库配置

#### 创建数据库
```sql
-- 创建数据库
CREATE DATABASE UniAdminDB;
GO

USE UniAdminDB;
GO

-- 执行初始化脚本
-- 路径: database/init_schema.sql
```

#### 配置连接字符串
在 `ProjectConfig.json` 中配置：
```json
{
  "database": {
    "driver": "MSSQL",
    "server": "localhost",
    "database": "UniAdminDB",
    "username": "sa",
    "password": "your_password",
    "integratedSecurity": false
  }
}
```

### 4. IDE 配置

#### 代码格式化
- Tools → Options → Editor Options → Auto Indent
- Tools → Options → Formatter → Delphi Formatter

#### 编译选项
- Project → Options → Building → Delphi Compiler
  - Optimization: On
  - Stack frames: Off
  - Debug information: Debug (开发时), None (发布时)

#### 路径配置
确保以下路径在 Library Path 中：
- UniGUI 源码路径
- 项目第三方库路径

### 5. 验证环境

#### 编译项目
1. 打开 `UniAdmin.dproj`
2. Build → Compile Project (Ctrl+F9)
3. 确保编译成功，无错误

#### 运行项目
1. Run → Run (F9)
2. 浏览器自动打开 `http://localhost:8077`
3. 使用默认账号登录（见 README.md）

## 代码规范

### 1. 命名规范

#### PascalCase (类名、接口名、单元名)
```delphi
TUserManager = class;           // 类
IUserDataAccess = interface;     // 接口
unit UserManager.pas;             // 单元
```

#### camelCase (变量名、方法名、属性名)
```delphi
var
  userName: string;
  userId: Integer;

function getUserById(id: Integer): TUser;
property IsActive: Boolean read FIsActive write FIsActive;
```

#### UPPER_CASE (常量、枚举)
```delphi
const
  MAX_RETRY_COUNT = 3;
  DEFAULT_TIMEOUT = 30000;

type
  TUserRole = (ADMIN, USER, GUEST);
```

#### T_ 前缀 (类型)
```delphi
type
  TUser = record;
  TUserList = array of TUser;
  TUserEvent = procedure(Sender: TObject; User: TUser) of object;
```

#### F_ 前缀 (私有字段)
```delphi
private
  FUserName: string;
  FUserId: Integer;
  FIsActive: Boolean;
```

#### A_ 前缀 (参数)
```delphi
procedure SaveUser(const AUserName: string; AUserId: Integer);
```

### 4. 异常处理

#### 标准异常处理
```delphi
try
  // 可能抛出异常的代码
  SaveToDatabase(Data);
except
  on E: EDatabaseError do
  begin
    LogError('数据库错误: ' + E.Message);
    raise;
  end;
  on E: Exception do
  begin
    LogError('未知错误: ' + E.Message);
    raise;
  end;
end;
```

#### 资源保护
```delphi
var
  Query: TFDQuery;
begin
  Query := TFDQuery.Create(nil);
  try
    Query.Connection := FDConnection;
    Query.SQL.Text := 'SELECT * FROM Users';
    Query.Open;
    // 使用 Query
  finally
    Query.Free;
  end;
end;
```

### 5. 性能优化

#### 避免频繁字符串拼接
```delphi
// 不推荐
Result := '';
for i := 0 to 1000 do
  Result := Result + IntToStr(i);

// 推荐
var
  SB: TStringBuilder;
begin
  SB := TStringBuilder.Create;
  try
    for i := 0 to 1000 do
      SB.Append(IntToStr(i));
    Result := SB.ToString;
  finally
    SB.Free;
  end;
end;
```

#### 使用接口自动管理生命周期
```delphi
var
  Manager: IUserManager;
begin
  Manager := TUserManager.Create;
  // 使用 Manager，离开作用域自动释放
end;
```

## 测试指南

### 1. 单元测试

#### 创建测试单元
```delphi
unit TestUserManager;

interface

uses
  System.SysUtils, DUnitX.TestFramework,
  App.Modules.User.UserManager;

type
  [TestFixture]
  TTestUserManager = class(TObject)
  private
    FUserManager: TUserManager;
  public
    [Setup]
    procedure Setup;
    [TearDown]
    procedure TearDown;

    [Test]
    [TestCase('ValidUser', '1,John,Doe')]
    [TestCase('InvalidId', '-1,John,Doe')]
    procedure TestGetUserById(Id: Integer; FirstName, LastName: string);

    [Test]
    procedure TestCreateUser;
  end;

implementation

procedure TTestUserManager.Setup;
begin
  FUserManager := TUserManager.Create;
end;

procedure TTestUserManager.TearDown;
begin
  FUserManager.Free;
end;

procedure TTestUserManager.TestGetUserById(Id: Integer; FirstName, LastName: string);
var
  User: TUser;
begin
  User := FUserManager.GetUserById(Id);

  if Id > 0 then
  begin
    Assert.IsNotNull(User, '用户不应为空');
    Assert.AreEqual(FirstName, User.FirstName, '名字不匹配');
    Assert.AreEqual(LastName, User.LastName, '姓氏不匹配');
  end
  else
  begin
    Assert.IsNull(User, '无效ID应返回空');
  end;
end;

procedure TTestUserManager.TestCreateUser;
var
  User: TUser;
begin
  User.FirstName := 'Test';
  User.LastName := 'User';
  User.Email := 'test@example.com';

  Assert.IsTrue(FUserManager.CreateUser(User), '创建用户应成功');
  Assert.Greater(User.Id, 0, '用户ID应大于0');
end;

initialization
  RegisterTest(TTestUserManager);

end.
```

#### 运行测试
- Tools → Test Runner
- 选择测试单元 → Run
- 查看测试结果和覆盖率

### 2. 集成测试

#### 测试数据库操作
```delphi
[TestFixture]
TTestUserIntegration = class(TObject)
private
  FConnection: TFDConnection;
public
  [Setup]
  procedure Setup;
  [TearDown]
  procedure TearDown;

  [Test]
  procedure TestDatabaseConnection;
  [Test]
  procedure TestUserCRUD;
end;

procedure TTestUserIntegration.Setup;
begin
  FConnection := TFDConnection.Create(nil);
  FConnection.DriverName := 'MSSQL';
  FConnection.Params.Values['Server'] := 'localhost';
  FConnection.Params.Values['Database'] := 'UniAdminDB_Test';
  FConnection.Params.Values['User_Name'] := 'sa';
  FConnection.Params.Values['Password'] := 'test';
  FConnection.Connected := True;
end;

procedure TTestUserIntegration.TearDown;
begin
  FConnection.Free;
end;

procedure TTestUserIntegration.TestDatabaseConnection;
begin
  Assert.IsTrue(FConnection.Connected, '数据库连接应成功');
end;

procedure TTestUserIntegration.TestUserCRUD;
var
  Query: TFDQuery;
begin
  Query := TFDQuery.Create(nil);
  try
    Query.Connection := FConnection;

    // Insert
    Query.SQL.Text := 'INSERT INTO Users (FirstName, LastName) VALUES (:FN, :LN)';
    Query.ParamByName('FN').AsString := 'Test';
    Query.ParamByName('LN').AsString := 'User';
    Query.ExecSQL;
    Assert.Greater(Query.RowsAffected, 0, '插入应成功');

    // Select
    Query.SQL.Text := 'SELECT * FROM Users WHERE FirstName = :FN';
    Query.ParamByName('FN').AsString := 'Test';
    Query.Open;
    Assert.IsTrue(Query.RecordCount > 0, '应找到记录');

    // Update
    Query.Edit;
    Query.FieldByName('LastName').AsString := 'Updated';
    Query.Post;

    // Delete
    Query.Delete;
  finally
    Query.Free;
  end;
end;
```

### 3. 测试最佳实践

- **AAA 模式**: Arrange (准备) → Act (执行) → Assert (断言)
- **独立性**: 每个测试应该独立，不依赖其他测试
- **可重复性**: 测试结果应该可重复，不受环境影响
- **命名规范**: Test + 方法名 + 场景 + 预期结果
- **覆盖边界**: 测试正常值、边界值、异常值

## 常见问题

### Q1: 编译时找不到 UniGUI 单元
**A**: 确保 UniGUI 已正确安装，并在 Project → Options → Delphi Compiler → Search Paths 中添加 UniGUI 路径。

### Q2: 运行时数据库连接失败
**A**: 检查 `ProjectConfig.json` 中的数据库配置，确保 SQL Server 服务正在运行，用户名密码正确。

### Q3: 插件无法加载
**A**: 检查插件目录路径，确保插件 DLL 文件存在，且与主程序版本兼容。

### Q4: 权限验证失败
**A**: 确保用户已登录，且具有相应权限。检查 RBAC 配置是否正确。
