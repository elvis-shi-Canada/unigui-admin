## 三、使用建议

### 开发场景映射

| 场景 | 推荐案例 |
|-----|---------|
| 后台管理系统 | MegaDemo, AllFeaturesDemo, DBAppDemo |
| 登录认证 | LoginForm, LoginForm Cookies, HTTP Basic Authentication |
| 数据展示 | Grid系列, DBDemo, HyperGrid系列 |
| 数据录入 | Form系列, DBLookupComboBox, GridEditors |
| 图表报表 | ChartDemo系列, FastReport系列, PivotGrid |
| 文件管理 | FileUpload系列, DownloadDemo |
| 移动端适配 | Touch目录下对应案例 |
| 实时通信 | WebSocket系列 |
| 地理位置应用 | GoogleMaps, UniMap, FSMap |

### 代码复用提示

1. **参考 MainModule 和 ServerModule** - 服务端配置
2. **参考 LoginForm** - 认证流程
3. **参考 DBDemo/MasterDetail** - 数据库操作模式
4. **参考 Grid 系列** - 表格最佳实践
5. **参考 ClientEvents 系列** - 前后端交互
